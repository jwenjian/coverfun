# Cover Fun

在线面试再也不紧张啦！

这个小软件会在所有窗口之上添加一个透明层， 用恋爱模拟游戏里的对话框样式让在线面试看起来像是你和面试官在谈恋爱， 他/她对你非常感兴趣， 认为你非常适合这份工作。

按下 Esc 键隐藏（最小化）, 按下鼠标左键可拖动窗口。

玩的开心（祝你面试成功）。
